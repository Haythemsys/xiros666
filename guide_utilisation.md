# Guide d'utilisation du site Occult Esoteric

## Introduction
Ce document présente le site web Occult Esoteric, un portail mystique et ésotérique inspiré du site "The Hawk 555". Le site a été conçu pour offrir une expérience immersive dans l'univers de l'occultisme, avec des sections dédiées à l'astrologie, la magie, le tarot et d'autres pratiques ésotériques.

## Structure des fichiers

- `index.html` - Version arabe (RTL) du site
- `index_fr.html` - Version française du site
- `index_en.html` - Version anglaise du site
- `css/styles.css` - Feuille de style principale
- `js/main.js` - Script JavaScript pour les animations et la gestion multilingue
- `images/` - Dossier contenant les images du site
- `site_structure.md` - Documentation détaillée de la structure du site
- `todo.md` - Liste des tâches accomplies pour le projet

## Fonctionnalités principales

### 1. Interface multilingue
Le site est disponible en trois langues : arabe (par défaut), français et anglais. La navigation entre les langues se fait via les boutons en haut à droite de chaque page.

### 2. Animation d'introduction
Une animation d'introduction avec des messages mystiques apparaît au chargement du site, créant immédiatement une ambiance ésotérique.

### 3. Navigation intuitive
Le menu principal permet d'accéder facilement aux différentes sections du site : Accueil, Astrologie, Magie & Rituels, Tarot & Divination, Actualités Mystiques, Bibliothèque Secrète et À Propos.

### 4. Design responsive
Le site s'adapte à tous les types d'écrans, des smartphones aux grands écrans d'ordinateur.

### 5. Effets visuels mystiques
Des effets de fumée, des symboles flottants et des transitions élégantes enrichissent l'expérience utilisateur et renforcent l'ambiance occulte.

## Personnalisation

### Modification du contenu
Pour modifier le contenu des sections, vous pouvez éditer les fichiers HTML correspondants à chaque langue. Les textes sont organisés de manière à faciliter les mises à jour.

### Ajout de nouvelles sections
Pour ajouter de nouvelles sections, suivez le modèle des sections existantes dans les fichiers HTML et ajoutez les traductions correspondantes dans le fichier `main.js`.

### Modification du style
Le fichier `styles.css` contient toutes les définitions de style. Les variables CSS en début de fichier permettent de modifier facilement les couleurs principales et les polices.

## Déploiement

Pour déployer le site sur un serveur web :
1. Téléchargez tous les fichiers sur votre serveur
2. Assurez-vous que la structure des dossiers est préservée
3. Configurez votre serveur pour utiliser `index.html` comme page d'accueil
